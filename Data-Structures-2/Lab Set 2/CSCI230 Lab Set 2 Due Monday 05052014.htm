Lab Set 2 (Due: Monday, 05/05/2014)

Problems:

Week 5: Implement quick select algorithm to find kth smallest element as discussed in lecture/book.  Test it with a few integer values.  Optional:  count the depth of recursive calls to confirm that it should be much smaller than logn.

Week 6: Holiday and exam

Week 7: Implement find/union partition structure (option 1 or option 2):

Option 1: a list-based approach as discussed in class/book.  

Option 2: a tree-based approach.  In Java, use Partition.java and Position.java in 7th edition, add a driver to to test it for n operations (make cluster/set, find, and union) and collect some data to confirm it would cost nlog*n for n operations.   In C++, use Partition.java and Position.java in 7th edition as a guide to create your own code, add a driver to to test it for n operations (make cluster/set, find, and union).  You can ignore the validate operation as long as you refer to a valid position.

Implement Brute Force Pattern Matching with BM's looking-glass heuristic.  Is it better than original Brute Force Pattern Matching?  Explain.

Week 8: Try BM and KMP pattern matching (code is given) on various T and P.  Use a driver to check them out. Make sure to understand the algorithms.

Perform Programming Project 12.6 (page 591) and it can be used as part of Project 3 E.C.  Assume alphabet consists of 26 lowercase letters and try S = {bear, bell, bid, bull, buy, sell, stock, stop}.

Week 9: Java -- Do both Matrix-Chain Product and LCS; C++ Only need to do one of the two problems, but feel free to try both.  For LCS, print out the longest common sub-sequence of the two strings.  For MCP, print out resulting matrix, minimum number of operations and discuss how the resulting matrix will help to determine the parentheses.  Feel free to try the last step out with your program.

************************* End of Lab Set 2 ************************************

Provide answers, input/output, and source code as applicable; submit one set if you are working with a lab partner.  Include a table of contents of all submitted items in correct order.  You can submit printouts of everything or you can submit electronically via MR.  You can earn up to 4 extra points for good work and an organized package.  For electronic submission, you must put everything in one PDF file (put in Word and save as PDF format) and submit this file via MR and a printout of table of contents indicating that a PDF has been submitted electronically.
